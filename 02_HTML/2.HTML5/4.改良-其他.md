# 改良-其他

- `<a>` 
    - 新增 download 属性
        - 记录下载到本地时，采用的文件名
    - media 和 type 可忽略

- `<ol>`
    - 新增 reversed 属性
        - 倒序显示

- `<script>`
    - 新增 defer 属性 (常用)
        - 表示: 此js加载完不会立即执行，要等整个HTML文档加载完(包括内部css+js)
    - 新增 async 属性
        - 表示: 此js加载完立即执行

- `<small>`
    - 更改用途为: 表示“小字印刷体”文字
        - 比如免责声明和版权声明

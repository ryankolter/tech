# vim - 光标移动

- 步进 移动
    - 左右移
        - `h` / `l` 
            - 光标 左/右 移一个字符
        - `数字` + `h` / `l`
            - 光标 左/右 移多个字符
    - 上下移
        - `j` / `k`
            - 光标 下/上 移一行
        - `数字` + `j` / `k`
            - 光标 下/上 移多行

- 单行 词间移动
    - `w`
        - 移到 下一个单词 的开头
    - `b`
        - 移到 上一个单词 的开头
    - `e`
        - 移到 当前单词 的结尾

- 极限 移动
    - `^`
        - 移到行头 
    - `$`
        - 移到行尾
    - `gg` 
        - 移到第一行 
    - `G`
        - 移到最后一行

- 翻页 移动
    - `Ctrl` + `b`
        - 向上翻页
    - `Ctrl` + `f`
        - 向下翻页

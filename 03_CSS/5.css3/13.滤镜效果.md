 # 滤镜效果

- 规则

    ```css
    filter: 取值;
    ```
    
- 参数可取
    - `brightness(百分比)`
        - 亮度滤镜
        - 0%-100% : 减弱
        - 大于100% : 增强

    - `contrast(百分比)`
        - 对比度滤镜
        - 0%-100% : 减弱
        - 大于100% : 增强

    - `saturate(百分比)`
        - 饱和度滤镜
        - 0%-100% : 减弱
        - 大于100% : 增强
    
    - `opacity(百分比)`
        - 透明度滤镜
        - 0% : 完全透明
        - 100% : 完全不透明
    
    - `blur(像素)`
        - 模糊度滤镜，即马赛克
        - 像素: 取值越大，越模糊

    - `grayscale(百分比)`
        - 灰度滤镜
        - 0% : 不做任何修改
        - 100% : 完全灰度，即黑白图片
        - 作用
            - 网站悼念一秒变灰
    
    - `sepia(百分比)`
        - 复古(褐色)滤镜
        - 0% : 不做任何修改
        - 100% : 完全复古
    
    - `hue-rotate(百分比)`
        - 色相旋转
        - 0deg : 不旋转
        - 360deg : 旋转360度
    
    - `drop-shadow(x-offset y-offset blur color)`
        - 阴影滤镜
        - 参考 阴影 box-shadow

# 双层flex嵌套问题

- 层次关系
    - 三条英文文字
        - RECENTS
        - OVERALL
        - MEMBERS
    - 两个按钮，包含
        - Live Class
        - 设置

- 【问题】内容溢出屏幕宽度
    
    ![w300](20221116_134927067.png "20221116_134927067")
    
    - 解决办法:
        - 把“三组英文文字”的文字区域，设置`overflow:scroll`
    
- 【设置后】不溢出了，但又导致文字重叠

    ![w300](20221116_134949137.png "20221116_134949137")

    - 对于这种双层flex嵌套，目前没办法解决
    - 只能放弃文字区域的flex，改成
        - 对文字区域整体使用`overflow:scroll`
        - 对文字项使用`display:inline-block`

# 浏览结构 ls/tree

- #### `ls` 罗列文件
    - `ls -l`
        - 有些系统可以用别名`ll`替代
        - 默认 以字母排序
        - 参数解释
            - `-l` 长格式，列出各种详细信息，比如权限、所有者、大小、修改时间
        - 拓展
            - `-la` 显示隐藏文件夹
            - `-lh` 以人类可读的方式显示大小
            - `-lF` 对于文件夹会在最后加上`\`做区分（一般用不上，因为大多终端会对文件夹自动高亮）
        - 排序相关
            - `-lt` 以修改时间排序（由近到远）
            - `-lS` 以文件大小排序（由大到小）
            - `r` 反序排列
    - `ls ~ /usr`
        - 罗列多个目录(这里是主目录和usr目录)
            

- #### `tree` 树状图列出目录内容
    - 需要安装
        ```bash
        yum install -y tree
        ```
        ```bash
        apt install -y tree
        ```

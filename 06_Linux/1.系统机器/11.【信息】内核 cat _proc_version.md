# 【信息】内核 cat /proc/version

- #### Ubuntu / Debian

    ```bash
    cat /proc/version
    ```
    - 输出
        ```ruby
        Linux version 5.4.0-29-generic (buildd@lgw01-amd64-035) ...
        ```

- #### Red Hat / Centos / Fedora
    
    ```bash
    cat /proc/version
    ```
    - 输出
        ```ruby
        Linux version 3.10.0-1160.42.2.el7.x86_64 ...
        ```
